## Install application on local machine

Clone repository and install npm packages
```zh
git clone https://github.com/martinsagat/app-optimization
cd app-optimization
npm install
```

Run in development env (http://localhost:9000)
```zh
npm run dev
```
Build for production
```zh
npm run build
```
